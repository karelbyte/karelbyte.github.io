# Portfolio Personal - Karel Puerto Díaz

Página personal moderna construida con React, Vite y Tailwind CSS.

## 🚀 Características

- Diseño moderno con gradientes y animaciones
- Navegación suave entre secciones
- Responsive design
- Efectos hover interactivos
- Tema oscuro con acentos púrpura/rosa

## 📦 Instalación

```bash
npm install
```

## 🛠️ Desarrollo

```bash
npm run dev
```

## 🏗️ Build

```bash
npm run build
```

## 📝 Personalización

Edita los componentes en `src/components/` con tu información personal:

- `Hero.jsx` - Información principal y enlaces sociales
- `About.jsx` - Descripción personal
- `Experience.jsx` - Experiencia laboral
- `Skills.jsx` - Habilidades técnicas
- `Projects.jsx` - Proyectos destacados
- `Contact.jsx` - Información de contacto

## 🎨 Tecnologías

- React 18
- Vite
- Tailwind CSS
- Lucide React (iconos)
